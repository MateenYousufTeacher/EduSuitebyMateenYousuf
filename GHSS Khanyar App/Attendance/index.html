      🚀 Quick Guide
      📋 Attendance Tab — Select class & date, mark each student Present or Absent, then tap Save.
      👩‍🎓 Students Tab — Add or remove students per class. Click names to edit them.
      📊 Reports Tab — Generate monthly summary reports per class.
      📤 Export Tab — Export attendance as text file or print view.
      🌙 Dark Mode — Toggle using the button in the header.
