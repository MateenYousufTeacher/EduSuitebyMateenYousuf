📖 Setup & Usage Guide
1
Save Your Files
Create a folder jkbose-school-app. Place index.html, manifest.json, service-worker.js, author.jpg, principal.jpg inside it.
2
Run Locally
VS Code: Install "Live Server" extension → right-click index.html → Open with Live Server.
Python: python -m http.server 8080 → visit http://localhost:8080
3
Install as App
Android Chrome: Menu → "Add to Home Screen".
Desktop Chrome: Click ⊕ icon in address bar → Install.
4
Host for Free
Netlify: Drag folder to netlify.com/drop — instant URL!
GitHub Pages: Push to GitHub repo → Settings → Pages.
Cloudflare Pages: pages.cloudflare.com
5
Workflow
1️⃣ Create Exam → 2️⃣ Select exam in "Enter Marks" → Choose preset students → Fill marks → Save → 3️⃣ View Results tab for auto-calculated grades and cards.
