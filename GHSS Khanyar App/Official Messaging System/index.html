📖 Setup & Usage Guide
1

📁 Save Your Files

Create a folder named jkbose-school-app on your computer. Inside it, place these files: index.html, manifest.json, service-worker.js, author.jpg, principal.jpg.

2

🌐 Run Locally

Option A – VS Code: Install the "Live Server" extension, right-click index.html, and click "Open with Live Server".
Option B – Python: Open terminal in folder, run python -m http.server 8080, then visit http://localhost:8080.

3

📱 Install the App

On Android (Chrome): Tap the three-dot menu → "Add to Home Screen". On Desktop (Chrome): Look for the install icon (⊕) in the address bar and click it.

4

☁️ Host for Free

GitHub Pages: Push folder to GitHub repo → Settings → Pages → Deploy from branch.
Netlify: Drag & drop your folder at netlify.com/drop — instant live URL, free forever.
Cloudflare Pages: Connect GitHub repo at pages.cloudflare.com.

5

💾 Data Storage

All messages are saved automatically in your browser's LocalStorage. Data persists even after closing the browser. No internet needed after first load.

6

🔕 Offline Mode

Once visited, the app works fully offline via the Service Worker cache. No internet needed to send or view messages.
